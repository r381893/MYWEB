# MYWEB 庫洛米留言板

這是一個基於 Firebase 的即時留言板，具有可愛的庫洛米主題。

## 如何執行 (How to Run)

由於這是一個靜態網頁專案，您不需要安裝任何伺服器環境。

### 方法 1：直接開啟
1. 在資料夾中找到 `index.html`。
2. 雙擊檔案，使用瀏覽器開啟即可。

### 方法 2：終端機指令
如果您使用 Mac，可以在終端機輸入：
```bash
open index.html
```

## 功能說明
- **[新功能] 暱稱系統**：在留言框上方輸入您的暱稱，系統會自動記住，下次不用再輸入。
- **留言**：輸入文字並送出，支援 Firebase 即時更新。
- **貼圖**：點擊上方的表情符號按鈕，可快速插入貼圖。
- **深色模式**：點擊右上角的月亮/太陽圖示切換主題。
